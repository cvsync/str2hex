str2hex
=======

1-Liner string -> hexdecimal array converter

## Usage:

% echo "A9993E36 4706816A BA3E2571 7850C26C 9CD0D89D" | ./str2hex

0xa9, 0x99, 0x3e, 0x36, 0x47, 0x06, 0x81, 0x6a,

0xba, 0x3e, 0x25, 0x71, 0x78, 0x50, 0xc2, 0x6c,

0x9c, 0xd0, 0xd8, 0x9d,
